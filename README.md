# Abhilash Francis — Portfolio

Personal portfolio website showcasing my work as a Front-End Developer with 4+ years of experience building high-performance web applications with React.js, Next.js & TypeScript.

## Tech Stack

- **Framework:** React 18
- **Animations:** Framer Motion, react-type-animation
- **UI Effects:** react-parallax-tilt, react-countup
- **Styling:** CSS custom properties, dark/light theme toggle
- **Hosting:** Vercel

## Features

- Dark & light theme with persistent toggle
- Animated intro loading screen
- Typewriter effect on hero role line
- Scroll-reveal animations on all sections
- Proficiency bars that animate on scroll with counting numbers
- 3D tilt effect on project cards with shimmer on hover
- Custom glowing cursor (desktop only)
- Floating label inputs in contact form
- Frosted glass navbar with active section indicator
- Drifting gradient orbs and subtle noise overlay
- Full `prefers-reduced-motion` accessibility support

## Getting Started

```bash
npm install
npm start
```

Opens at [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
